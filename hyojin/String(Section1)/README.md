## String
### 문자열 메서드
- toUpperCase() : 문자열을 대문자로 바꾸는 메서드
- toLowerCase() : 문자열을 소문자로 바꾸는 메서드
- toCharArray() : 문자열을 배열로 바꾸는 메서드
### Character 클래스 메서드
- toUpperCase() : 대문자로 바꾸는 메서드
- toLowerCase() : 소문자로 바꾸는 메서드
- isLowerCase() : 소문자인지 확인하는 메서드
- toUpperCase() : 대문자인지 확인하는 메서드
### 아스키 코드
- 대문자 : 65~90
- 소문자 : 97~122
