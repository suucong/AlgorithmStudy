## 문제

###  조합 구하기

#### 설명
```
1부터 N까지 번호가 적힌 구슬이 있습니다. 이 중 M개를 뽑는 방법의 수를 출력하는 프로그램을 작성하세요.
```

#### 입력
```
첫 번째 줄에 자연수 N(3<=N<=10)과 M(2<=M<=N)이 주어집니다.
```

#### 출력
```
첫 번째 줄에 결과를 출력합니다.
출력 순서는 사전순으로 오름차순으로 출력합니다.
```

#### 예시 입력
- 입력
    ```
  4 2
    ```
- 출력
    ```
  1 2
  1 3
  1 4
  2 3
  2 4
  3 4
  ```
